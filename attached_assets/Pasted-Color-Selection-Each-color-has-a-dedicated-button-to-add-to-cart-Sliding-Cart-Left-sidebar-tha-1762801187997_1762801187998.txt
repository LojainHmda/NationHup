🔧 INTEGRATION INSTRUCTIONS
1. Data Structure
Product Format:

{
  sku: 'JS0375',
  name: 'Premium Crew Sweatshirt',
  price: 85.67,
  originalPrice: 119.90, // optional
  colors: [
    { name: 'Black', hex: '#000000', imageUrl: '/path/to/black.png' },
    { name: 'White', hex: '#FFFFFF', imageUrl: '/path/to/white.png' }
  ],
  sizes: ['XS', 'S', 'M', 'L', 'XL', 'XXL'],
  availableSizes: {
    'Black-XS': 5,
    'Black-S': 10,
    'White-XS': 7,
    // ... (color-size keys with stock quantities)
  }
}

Cart Product Format:

{
  id: 'JS0375-Black', // sku-color
  sku: 'JS0375',
  name: 'Premium Crew Sweatshirt',
  color: 'Black',
  imageUrl: '/path/to/black.png',
  price: 85.67,
  sizes: ['XS', 'S', 'M', 'L', 'XL', 'XXL'],
  quantities: { 'XS': 0, 'S': 2, 'M': 0, ... }, // size: quantity
  availableSizes: { 'XS': 5, 'S': 10, ... } // size: stock
}

2. Dependencies Required
Install these shadcn/ui components:

npx shadcn@latest add button
npx shadcn@latest add input
npx shadcn@latest add checkbox
npx shadcn@latest add toast

Install lucide-react icons:

npm install lucide-react

3. API Endpoints to Create
Replace the mock data with these API calls:

GET /api/products - Fetch all products
POST /api/cart/add - Add product color variant to cart

{ "sku": "JS0375", "color": "Black" }

PATCH /api/cart/:productId/quantity - Update quantity

{ "size": "M", "quantity": 5 }

DELETE /api/cart/:productId - Remove product from cart

4. Quick Start
Copy all 5 files to your project
Update image imports in home.tsx with your product images
Replace mock products array with API call: useQuery({ queryKey: ['/api/products'] })
Replace cartProducts state with API call: useQuery({ queryKey: ['/api/cart'] })
Update handlers to use API mutations instead of local state

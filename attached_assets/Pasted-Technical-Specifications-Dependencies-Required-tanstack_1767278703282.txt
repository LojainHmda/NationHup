Technical Specifications
Dependencies Required
{
  "@tanstack/react-query": "^5.x",
  "wouter": "^3.x",
  "lucide-react": "^0.x",
  "xlsx": "^0.x"
}
Features
Dynamic Pivot Engine - Drag-and-drop dimension nesting
10 Grouping Dimensions - Cart Source, Brand, Size, Age Group, Category, Color, Main Category, Model, SKU, Product Name
Recursive Hierarchical Grouping - Unlimited nesting depth
Aggregated Metrics - Volume (quantity) and Aggregate Value (total price)
Excel Export - Flattens hierarchy with full path
Search - Filter by product name, SKU, brand, size, or cart
Expandable Tree Table - Visual indentation with color-coded depth indicators
Data Requirements
The component expects cart data with this structure:

interface CartItem {
  productId: string;
  productName: string;
  sku: string;
  brand: string;
  size: string;
  quantity: number;
  unitPrice: number;
  totalPrice: number;
  color?: string;
}

<Route path="/cart-hierarchy" component={CartHierarchyPage} />

const parseNumber = (val: any): number => {
  if (typeof val === 'number') return val;
  if (typeof val === 'string') {
    const parsed = parseFloat(val);
    return isNaN(parsed) ? 0 : parsed;
  }
  return 0;
};
